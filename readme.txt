# WoBBoB Official Website

## Structure
- `index.html` - Main landing page
- `about-us.html` - Information about WoBBoB
- `trade-a-them-bot.html` - Details about the Trade A Them Bot
- `bobbob.html` - Details about The BoBBoT AI chatbot
- `gallery.html` - Image gallery
- `contact-us.html` - Contact information and social media links
- `articles.html` - Collection of music-related articles
- `article.html` - Individual article page
- `sitemap.html` - Website navigation map

## Folders
- `css/` - Contains stylesheets
- `js/` - Contains JavaScript files
- `images/` - Contains image assets

## Deployment
1. Upload all files to GitHub
2. Deploy using Vercel
3. Ensure all links are correctly mapped
